## Alarm Zones and Loops

A zone is a group of sensors connected to the alarm panel.

### Common Zones
- Zone 1: Main Door (Magnetic Contact)
- Zone 2: Living Room (PIR Motion Sensor)
- Zone 3: Yard (Outdoor Beam)

### Loops
- Normally Closed (NC)
- End of Line Resistor (EOL)
## Alarm Sensors

### PIR Motion Sensor
Detects movement using infrared heat changes.

### Magnetic Contact
Used on doors and windows.

### Beam Sensor
Outdoor detection using infrared beams.
